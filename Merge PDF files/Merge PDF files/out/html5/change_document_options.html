Change Document Options

Change the document options before publishing one PDF file.

The task is optional.

  1. Select Document Options.
    The Document Options window opens.
  2. Under Save PDF with select Color.
  3. Select PDF Options.
    The PDF Options window opens.
  4. Select Include project info in PDF.
  5. Select Export layers to PDF.
  6. Select OK.
